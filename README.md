# Physical AI: Embodied Intelligence in the Real World

![Physical AI Concept](/img/physical-ai/embodied-intelligence.jpg)

This repository contains the complete open textbook "Physical AI: Embodied Intelligence in the Real World" created as part of a hackathon project.

## Book Structure

The textbook is built using Docusaurus and contains comprehensive chapters covering all aspects of Physical AI:

- Introduction
- Embodied Intelligence
- ROS2 Fundamentals
- Gazebo Simulation
- Isaac Sim
- URDF/XACRO Modeling
- Synthetic Data Generation

## Building the Book

To build and serve the book locally:

```bash
# Install dependencies
npm install

# Serve the book locally
npm start

# Build for production
npm run build
```

The book will be available at http://localhost:3000

## Project Status

This textbook represents a complete and comprehensive resource on Physical AI, covering both theoretical foundations and practical applications. All chapters include detailed explanations, mathematical formulations, code examples, and exercises.
