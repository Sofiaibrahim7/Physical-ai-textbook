# Physical AI: Embodied Intelligence in the Real World

This repository contains the complete open textbook "Physical AI: Embodied Intelligence in the Real World" created as part of a hackathon project.

## Book Structure

The textbook is built using mdBook and contains 14 comprehensive chapters covering all aspects of Physical AI:

1. Introduction
2. Embodiment and Situated Cognition
3. Physics for AI
4. Sensors and Perception
5. Motor Control
6. Reinforcement Learning in Physical Domains
7. Sim-to-Real Transfer
8. Learning from Demonstration and Imitation
9. Multi-Agent Physical AI and Interaction
10. Safety and Robustness in Physical AI
11. Energy Efficiency and Sustainable Physical AI
12. Case Studies and Real-World Applications
13. Future Directions and Open Challenges
14. Conclusion

## Building the Book

To build and serve the book locally, you need to have mdBook installed:

```bash
# Install mdBook
cargo install mdbook

# Serve the book locally
mdbook serve --open
```

The book will be available at http://localhost:3000

## Project Status

This textbook represents a complete and comprehensive resource on Physical AI, covering both theoretical foundations and practical applications. All chapters include detailed explanations, mathematical formulations, code examples, and exercises.
