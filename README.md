# Physical AI Textbook

Welcome to the Physical AI Textbook project! This comprehensive resource covers the fundamentals and advanced concepts of Physical AI and Humanoid Robotics.

## Project Structure

- `frontend/` - Docusaurus-based documentation website
- `backend/` - Python backend services
- `spec/` - Project specifications and templates
- `specs/` - Detailed project specifications

## Frontend Setup

The frontend is built with Docusaurus, a modern static website generator. To run locally:

```bash
cd frontend
npm install
npm run start
```

## Backend Setup

The backend is built with Python. To set up:

```bash
cd backend
pip install -r requirements.txt  # if there's a requirements.txt
```

## Deployment

This project is configured for deployment on Vercel. Check the `vercel.json` file for configuration details.
